## 📸 About

This is my personal developer portfolio showcasing my skills, projects, and certifications.
Built from scratch with a focus on clean design, smooth animations, and full responsiveness.

---

## ✨ Features

- 🌑 Dark theme with animated glow blobs and noise texture
- 🎯 Hero section with live stats (ML accuracy, CGPA, certifications)
- 🛠️ Skills grid with hover animations
- 📂 Featured project card with live accuracy bar visualization
- 🏅 Certifications list with icons
- 📬 Contact section with direct links
- 📱 Fully responsive across desktop and mobile
